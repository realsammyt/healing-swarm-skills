# cyberdelic-research

**Trigger:** `/cyberdelic-research`

Deep research into consciousness technologies and endo-technology science. Covers endo-technology evidence base, archaeoacoustic frequency research, consciousness assessment models, and altered states vs. traits literature. Parallel research stages for efficiency.

## Agents

- **Hyperhumanism Researcher** - Core research and Smith's framework
- **Traditions Scholar** - Historical and cultural context
- **Clinical Researcher** - Clinical evidence assessment
- **Ethics Guardian** - Evidence standards and claim review

## Inputs

| Input   | Required | Description                                                    |
| ------- | -------- | -------------------------------------------------------------- |
| `topic` | Yes      | Research focus (e.g., "endo-technology evidence base")         |
| `depth` | No       | `quick` (2-3 pages), `standard` (5-8), `comprehensive` (10-15) |

## Outputs

- `research-brief.md` - Integrated research document
- `evidence-synthesis.md` - Cross-domain evidence analysis
- `technology-assessment.md` - Endo-technology evaluation

## Examples

**Endo-technology evidence overview:**

```bash
/cyberdelic-research "endo-technology evidence base" --depth standard
```

**Comprehensive archaeoacoustic review:**

```bash
/cyberdelic-research "archaeoacoustic consciousness effects" --depth comprehensive
```

**Quick consciousness models summary:**

```bash
/cyberdelic-research "consciousness assessment models" --depth quick
```

| Domain                    | Key Sources                                            |

## Workflow

Research stages run in **parallel** for efficiency:

```
intake
  ├── primary_research (hyperhumanism-researcher)
  ├── traditions_research (traditions-scholar)      [parallel]
  └── clinical_context (clinical-researcher)
      ↓
  synthesis (hyperhumanism-researcher)
      ↓
  ethics_review (ethics-guardian)
```

## Evidence Standards

All research output must:

- Support all claims with citations
- Note sample sizes for empirical studies
- Label evidence levels (theoretical / preliminary / moderate / well-established)
- State limitations for every finding
- Use "suggests" language, never "proves"
- Distinguish Smith's theoretical work from underlying research

## Quality Gates

Before output is finalized:

- Multiple research streams integrated coherently
- Contradictions between sources acknowledged
- Evidence gaps identified
- Recommendations grounded in evidence level
- Pharmaco-technology exclusion maintained
- Primary sources preferred over secondary
- Peer-reviewed sources prioritized
