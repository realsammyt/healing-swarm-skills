# holotechnica-stack

**Trigger:** `/holotechnica-stack`

Personalized endo-technology experience stacks combining individually researched practices into designed consciousness recipes targeting altered traits. **NO pharmaco-technologies (HARD EXCLUSION).**

## Agents

- **Hyperhumanism Researcher** - Per-ingredient evidence and altered traits literature
- **Holotechnica Architect** - Stack design, timing, and contraindication logic
- **Content Writer** - Recipe language and framing
- **Ethics Guardian** - Safety, contraindications, and pharmaco-exclusion verification

## Inputs

| Input      | Required | Description                                                 |
| ---------- | -------- | ----------------------------------------------------------- |
| `target`   | Yes      | Trait target (e.g., "focused attention", "sustained calm")  |
| `depth`    | No       | `simple` (15 min), `standard` (30 min), `extended` (45 min) |
| `duration` | No       | Total duration in minutes                                   |

## Outputs

- `holotechnica-recipe.md` - Complete stack recipe with timing
- `ingredient-matrix.md` - Evidence levels per ingredient
- `integration-journal.md` - Reflection and trait-tracking prompts
- `evidence-brief.md` - Per-ingredient and stack-level evidence

## Examples

**Simple morning activation:**

```bash
/holotechnica-stack "morning activation" --depth simple --duration 15
```

**Standard focus stack:**

```bash
/holotechnica-stack "focused attention" --depth standard
```

**Extended deep exploration:**

```bash
/holotechnica-stack "deep exploration" --depth extended
```

## Safety: Additive Contraindication Logic

**Any ingredient flagged = entire stack flagged.** All contraindications from all ingredients combine (union, not intersection). If any single ingredient is contraindicated, the stack must be modified.

## Research Basis

> **Evidence level: Theoretical (stacking) / Varies (ingredients)**
>
> Individual endo-technologies have varying evidence bases (noted per ingredient).
> The Holotechnica stacking methodology — combining practices into designed sequences
> — is a theoretical framework developed by Carl Hayden Smith. No clinical evidence
> exists for specific combinations. The altered traits framework draws on Goleman &
> Davidson (2017).
