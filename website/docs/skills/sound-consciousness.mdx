# sound-consciousness

**Trigger:** `/sound-consciousness`

Museum of Consciousness-style audio experiences with short consciousness samples and Milliere 6D self-assessment between each sample.

## Agents

- **Hyperhumanism Researcher** - Consciousness models and assessment research
- **Archaeoacoustic Guide** - Consciousness sample design and sound journey
- **Content Writer** - Experience language and framing
- **Ethics Guardian** - Dissociation safety and evidence claims review

## Inputs

| Input      | Required | Description                                 |
| ---------- | -------- | ------------------------------------------- |
| `focus`    | Yes      | Sample focus (e.g., "mindfulness", "sound") |
| `duration` | No       | `20` (standard) or `30` (extended) minutes  |
| `samples`  | No       | Number of consciousness samples (3-6)       |

## Outputs

- `consciousness-exhibit.md` - Complete guided experience
- `self-assessment-form.md` - Milliere 6D tracking sheets
- `evidence-brief.md` - Research context with limitations

## Examples

**Standard mindfulness-focused session:**

```bash
/sound-consciousness "mindfulness samples" --duration 20
```

**Extended introspection journey:**

```bash
/sound-consciousness "introspection journey" --duration 30
```

## Session Structure

| Phase      | Duration     | Activity                          |
| ---------- | ------------ | --------------------------------- |
| Opening    | 3-4 min      | Safety briefing, baseline 6D      |
| Per Sample | 5-7 min each | Sample + grounding + 6D + journal |
| Closing    | 3-4 min      | Final grounding, comparison       |

## Milliere 6D Dimensions

Self-assessment after each consciousness sample (1-7 scale):

1. **Sense of Agency** - Control over your actions
2. **Sense of Ownership** - Body feeling like yours
3. **Self-Location** - Where YOU are in space
4. **First-Person Perspective** - Single viewpoint experience
5. **Unity of Experience** - Unified vs. fragmented
6. **Narrative Self** - Presence of your life story

## Research Basis

> **Evidence level: Preliminary (framework) / Varies (individual samples)**
>
> The Museum of Consciousness framework was developed by Carl Hayden Smith
> at Balliol College, Oxford. The Milliere 6D model (2017) provides a
> research-supported framework for self-assessment of consciousness
> dimensions. Individual sample types have varying evidence bases.

## Quality Gates

Before output is finalized:

- Dissociation screening included
- Grounding anchor between every sample
- 6D dimensions clearly explained with "no right answer" framing
- Baseline measurement at start for comparison
- Return-to-baseline protocol at end

- Medical disclaimer and professional referral information
