## Skills Overview

### Development Skills (Build New)

| Skill | Trigger | Purpose |
|-------|---------|---------|
| [Research](./research.mdx) | `/healing-research` | Deep research into healing traditions and clinical evidence |
| [Design](./design.mdx) | `/healing-design` | Design healing application UX and visuals |
| [Content](./content.mdx) | `/healing-content` | Create prayers, evidence boxes, visualizations |
| [Build](./build.mdx) | `/healing-build` | Develop accessible, privacy-first applications |
| [Review](./review.mdx) | `/healing-review` | Quality review (ethics, clinical, cultural, a11y) |
| [Deploy](./deploy.mdx) | `/healing-deploy` | Deploy and maintain applications |
| [Swarm](./swarm.mdx) | `/healing-swarm` | Full orchestrated development lifecycle |

### Integration Skills (Upgrade Existing)

| Skill | Trigger | Purpose |
|-------|---------|---------|
| [Integration](./integration.mdx) | `/healing-integrate` | Add healing properties to any existing project |

**Specialized Integration Patterns:**

| Pattern | Best For |
|---------|----------|
| [Project Analysis](/docs/integration/project-analysis) | AI-powered HEAL assessment of any codebase |
| [Healing Gamification](/docs/integration/healing-gamification) | Non-manipulative engagement mechanics |
| [Narrative Healing](/docs/integration/narrative-healing) | Interactive fiction and visual novels |
| [Game Healing](/docs/integration/game-healing) | Video game wellness mechanics |
